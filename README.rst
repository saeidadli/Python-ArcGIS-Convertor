Python ArcGIS convertor
***********************

A package to convert data between common formats used in python scripting to ESRI goedatabase. The fomats are including but not limited to:
	- geojson
	- Excel
	- GTFS
	- Geopandas GeoDataFrame
	- Pandas DataFrame
